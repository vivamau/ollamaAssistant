Ollama Assistant

A companion for Ollama that helps you get the most out of your local LLMs.

Checkout the code on Github and have fun!

Why Ollama Assistant

Ollama, along with LM Studio, was one of the first tools that made it easy to use AI on our computers. Even though Ollama has recently added a graphical interface to make it easier to use, some parts still feel a bit lacking. That’s why I’ve created an alternative interface with features that I think would be handy for everyday users.

Ollama Assistant is designed to be a lightweight and user-friendly application that enhances your experience with local LLMs. It provides a range of features, including prompt management, chat history, and customizable settings, all aimed at making your interactions with language models more efficient and enjoyable.

What is Ollama Assistant?


Ollama Assistant lets you save and organize prompts and link them together with your LLMs, makes it a breeze to manage the LLMs installed, and lets you create LLM‑boosted setups with documents (PDF, txt, Excel, HTML, markdown, etc.) or even with websites' content.

It is a web app built with NodeJS and ReactJS and SQLlite and it depends on a Ollama installation. You can turn it into a desktop app with Electron, just follow the instructions on the code repository.

Ollama Assistant is open source, published under the AGPL 3 license.
